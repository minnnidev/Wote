# Wote

### 📱 청소년들의 소비 고민을 도와주는 투표 커뮤니티 앱
청소년들이 또래 친구들과 함께 소비 고민을 나누어 합리적인 소비를 할 수 있도록 돕고, 타인의 소비 성향을 확인하며 본인의 소비 가치관을 확립할 수 있는 커뮤니티 플랫폼 앱입니다.
